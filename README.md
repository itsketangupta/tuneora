# 🎵 Tuneora

Tuneora is a modern music web app built by **Ketan Gupta**.  
It allows users to explore trending songs, discover popular artists, and enjoy music previews in a clean and simple interface.

---

## 🚀 Features

- 🔥 **Real-time Trending Songs**
  - Stay updated with the latest trending tracks.

- 🎤 **Popular Artists**
  - Discover top and trending artists easily.

- 🕒 **Recently Played**
  - Tracks you listen to are automatically saved.

- 🔍 **Search Functionality**
  - Search for songs quickly and easily.

- ▶️ **Music Preview**
  - Play songs with a **30-second preview**.

- ❤️ **Favorites System**
  - Add songs to your favorite list.

- 💾 **Local Storage Support**
  - Recently played and favorite songs are stored in browser local storage.

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- API Integration (for music data)

---

## 📂 How It Works

- Songs are fetched using an API.
- Users can:
  - Search songs
  - Play previews
  - Add/remove favorites
- Data like favorites and recently played songs are saved using **localStorage**, so they persist even after refreshing.
